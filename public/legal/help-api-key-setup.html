API Key Setup

Learn how to add a valid official Groq API key and use ASK-GPT properly.

Using an API Key

API key setup is important for using ASK-GPT properly.

To use the built-in Groq model options in ASK-GPT, you must add a valid official Groq API key from your own Groq account.

Important Requirements

  • Use an official Groq API key from your own Groq account.
  • Make sure the key is active and valid.
  • Make sure the selected model options are available for that key.
  • Enter the key correctly during setup.

If the Key Does Not Work

If the key is invalid, unsupported, expired, or entered incorrectly, some model options or supported features may not work correctly.

Important

Please watch the API key setup video carefully and follow the full process step by step. If you follow the process correctly and add a valid official Groq API key, you will be able to use ASK-GPT and its supported features properly.

API Key ব্যবহার

ASK-GPT সঠিকভাবে ব্যবহার করার জন্য API key setup গুরুত্বপূর্ণ।

গুরুত্বপূর্ণ শর্ত

  • আপনার নিজের Groq account-এর official Groq API key ব্যবহার করুন।
