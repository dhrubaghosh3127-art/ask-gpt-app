Getting Started with ASK-GPT

ASK-GPT is designed to help you with AI chat, writing support, study help, translation, coding help, voice input, and image-based analysis.

You can use ASK-GPT by typing a message, speaking with voice input, or uploading an image for analysis.

Important

ASK-GPT currently supports image analysis only. It does not support AI image generation or photo creation.

Voice Input

ASK-GPT supports voice input so you can speak instead of typing.

Voice recognition quality may depend on microphone quality, internet connection, background noise, speaking clarity, and the selected input language.

If voice input does not work properly, try again in a quieter environment and make sure microphone permission is enabled.

Image Upload and Analysis

ASK-GPT supports image upload for analysis.

You can upload screenshots, notes, text images, and other visible content for AI assistance.

The app analyzes the visible content in the uploaded image and responds based on what is shown.

Important

ASK-GPT supports image understanding and image analysis. It does not support image generation or photo creation.

API Key Setup

API key setup is important for using ASK-GPT properly.

To use the built-in Groq model options in ASK-GPT, you must add a valid official Groq API key from your own Groq account.

If the key is unsupported, invalid, expired, or entered incorrectly, some model options may not work correctly.

Please watch the API key setup video carefully and follow the full process step by step. If you complete the setup correctly and add a valid official Groq API key, you will be able to use ASK-GPT and its supported features properly.
